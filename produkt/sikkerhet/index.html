Eksempel Produkt

Sikkerhet og etterlevelse

Dette er et eksempel på hvordan vi kan beskrive sikkerhet i Scope. Vi fremhever både tekniske kontroller, rutiner rundt personvern og hva kunder kan forvente ved revisjoner.

Tekniske tiltak

Alle miljøer er containerisert og kjører i separate VPC-er med zero-trust tilnærming.

  • ✔ Kryptering i hvile (AES-256) og transport (TLS 1.3)
  • ✔ MFA på alle interne admin-verktøy
  • ✔ Ukentlige sikkerhetstester og automatiske avviksmeldinger

Personvern

Vi følger GDPR og lagrer kun data som er nødvendige for tjenesten. Avtaler på norsk eller engelsk.

  • ✔ Standard databehandleravtale inkludert
  • ✔ Data residency i EU/EØS
  • ✔ Retningslinjer for sletting innen 30 dager

Prosedyrer

Hendelseshåndtering er forankret i en fast playbook og varsler både internt og eksternt.

  • ✔ 24/7-varsling via Slack og telefon
  • ✔ Årlig beredskapsøvelse med kunderepresentanter
  • ✔ Tilgangsrevision hver 90. dag
